# ADCM Installer

### Build
```shell
make
```

### Usage
Run ADCM
```shell
# see `adcm up --help` command
adcm up
```

Stop ADCM
```shell
adcm down
```

Configure project
```shell
# see `adcm init --help` command
adcm init -i
# Enter PostgreSQL Login (default: adcm):
# Enter PostgreSQL Password:
adcm up
```
